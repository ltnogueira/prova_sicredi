Projeto de Automação de Testes Web 

Este projeto tem como objetivo automatizar os testes de funcionalidade da aplicação web, garantindo sua qualidade e evitando erros manualmente. 

Tecnologias Utilizadas 

• Java - Linguagem de programação utilizada para escrever os scripts de teste. 

• Selenium WebDriver - Biblioteca de automação de testes para aplicações web utilizadas para controlar o navegador e interagir com a aplicação. 

• JUnit - Framework de testes unitários usado para organizar e executar os testes.


Pré-requisitos 

Antes de executar os testes, certifique-se de ter as seguintes ferramentas instaladas em sua máquina: 

• Conjunto de Desenvolvimento Java (JDK) 

• IDE de sua preferência (Ex.: Eclipse, IntelliJ IDEA)


Como executar os testes 

• Clone este projeto para sua máquina local usando o comandogit clone https://github.com/LeonardoTeixeiraNogueira/prova_sicredi.git

• Abra o projeto em sua IDE preferida 

• Execute os testes usando o JUnit como corredor de testes. 

Estrutura do Projeto 

Este projeto está garantido da seguinte maneira: 

• src/main/java- Contém o código-fonte dos scripts de teste. 

• src/test/java- Contém os arquivos de teste JUnit.
